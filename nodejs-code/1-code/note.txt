ES6

块级作用域，凡事在{}中的代码集称为一个代码块
在代码块中声明的变量，只在当前块中生效，但是
ES5没有块级作用域。

let 关键字，用来声明变量，跟var类似，但是也有区别

let 在某一代码块中声明的变量，只在当前代码块中有效
相当于创建了块级作用域，其主要作用可以避免全局变量
污染

const 关键字，用来声明常量，ES5没有常量的概念
所谓的常量就是指一旦定义并初始化值后不可再修改

变量的解构赋值，按照一定的模块批量的声明变量

var [a, b, c] = [1, 2, 3]; // 数组方式

// 左边多
var [a, b, c, d] = [1, 2, 3]; // d 为 undefined

// 右边多
var [a, b, c] = [1, 2, 3, 4]; // 忽略多余的值

// ... 剩余的
var [a, b, ...c] = [1, 2, 3, 4]; // c [3, 4]
